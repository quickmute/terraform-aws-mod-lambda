# About
This is download package to help with building reports in Python:
- datautil (2.8.2)
- et_xmlfile (1.1.0)
- numpy (1.26.0)
- openpyxl (3.1.2)
- pandas (2.1.1)
- pytz (2023.3)
- pytzdata (2020.1)
- tzdata (2023.3)
- six (1.16.0)

Package build in Amazon-Linux-Python container image running Python 3.11
